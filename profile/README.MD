O **ViverBem** é um ecossistema mobile projetado para mitigar erros na medicação e promover a autonomia de idosos (60+). Através de visão computacional e inteligência artificial, o aplicativo transforma a câmera do smartphone em um assistente de saúde inteligente, integrado a uma infraestrutura robusta em nuvem.

---

## 🎯 O Problema
A dificuldade na leitura de bulas com letras pequenas, o esquecimento de horários e a confusão entre comprimidos semelhantes são causas frequentes de internações. O ViverBem ataca essa dor com foco absoluto em **UX (User Experience) para a terceira idade**.

## ✨ Funcionalidades Principais

* 🔍 **Identificação por IA:** Reconhecimento instantâneo de fármacos via Google Vision AI.
* 💊 **Bula Simplificada:** Tradução de termos técnicos para linguagem clara e acessível.
* ⏰ **Gestão de Doses:** Sistema de alarmes inteligentes com confirmação de ingestão.
* 🗣️ **Acessibilidade 360º:** Interface de alto contraste, fontes dinâmicas e leitura por voz (Text-to-Speech).
* 🆘 **Rede de Apoio:** Notificação automática para cuidadores ou familiares em caso de doses perdidas.

## 🛠️ Stack Tecnológica

| Camada | Tecnologia | Descrição |
| :--- | :--- | :--- |
| **Mobile** | `React Native + TypeScript` | App multiplataforma (Android/iOS) |
| **Backend** | `Node.js + Python` | API Rest e processamento de imagem |
| **IA** | `Google Vision AI` | OCR e reconhecimento de objetos |
| **Estilização** | `Tailwind CSS (NativeWind)` | Design responsivo e utilitário |
| **Design** | `Figma` | Prototipagem de alta fidelidade |
| **Organização** | `Trello` | Separar tasks e organizar entregas |
